Codeforces Clone
نظرة عامة عن المشروع:
منصة ويب وتطبيق يتيحان للمستخدمين حل مسائل برمجية، مع نظام متابعة للتقييم، وتصنيف، ومراقبة أداء، وإدارة من قبل الأدمن. تهدف إلى تحسين مهارات البرمجة بطريقة تنافسية وتعليمية

 الميزات الرئيسية (Features).
 ## ✨ الميزات الرئيسية

### 👤 المستخدمون
- تسجيل الدخول وإنشاء حساب جديد.
- ملف شخصي يعرض الإنجازات والنقاط.
- متابعة التقدم وحل المسائل.

### 📝 المسائل
- عرض قائمة المسائل مع التصنيفات (مستوى الصعوبة، النوع).
- إرسال الحلول ومراقبة النتائج.
- مراقبة وقت التكويد (Coding Session Tracking).

### 🛠️ الأدمن
- إضافة وتعديل وحذف المسائل.
- تقييم حلول المستخدمين.
- مراقبة نشاط المستخدمين وإدارة النظام.

### 📊 التقييم والتحفيز
- نظام نقاط وترتيب (Leaderboard).
- شارات تحفيزية (Badges) عند إنجاز مهام معينة.ة.

## 🚀 كيفية تشغيل المشروع محلياً

### المتطلبات الأساسية
- PHP >= 8.1
- Composer
- MySQL
- Node.js (لتشغيل Vite)

### خطوات التثبيت

1. استنساخ المستودع:
   ```bash
   git clone https://github.com/latifaabd616/codeforces-clone-laravel.git
   cd codeforces-clone-laravel
      2. الاعتماديات:
composer install
npm install
إ3.عداد ملف البيئة:
cp .env.example .env
ثم قم بتعديل بيانات قاعدة البيانات في ملف .env.
تشغيل الهجرات (Migratios) والبيانات الأولية:
php artisan migrate --seed
تشغيل السيرفر:
php artisan serve
افتح المتصفح على:http://localhost:8000
